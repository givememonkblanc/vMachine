**Actionability:**

- ~5% are likely data quality issues (zero-resource VMs)
- ~2% require infrastructure capability (UEFI + Secure Boot)

## 7. Known Limitations

- All data is synthetic (generated by `generate_benchmark_inventory.py` with seed=42)
- Real-world compatibility distributions will differ
- The IDE controller flag is a warning (high), not a critical blocker — actual OpenStack support depends on the deployment's `hw_disk_bus` configuration
- Unsupported OS detection is based on a hardcoded blocklist; real-world OS detection requires accurate `guest_os` strings from vCenter
